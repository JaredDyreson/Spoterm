# Spoterm

A Spotify command line client, inspired by [this reddit post](https://www.reddit.com/r/unixporn/comments/cfd64p/oc_new_cli_discord_client_discpp_that_is_purely/)

# Features

- go to particular artist
- list artists:
	- albums
	- song count
	- album count
